# 🌍 Dictionary.io – Smart Translator Tool

> A modern, feature-rich JavaScript translator app with speech, copy, dark mode, and smooth UI interactions — built with real-world functionality in mind.

---

## ✨ Features

- 🌐 **Multi-language Translation** (API powered)  
- 🔊 **Text-to-Speech** (listen to input & translated output)  
- 📋 **One-click Copy** for both text areas  
- 🔄 **Instant Swap** (text + selected languages)  
- 🌙 **Dark Mode Toggle** (clean glassmorphism UI)  
- 🔔 **Toast Notifications** (modern feedback instead of alerts)  
- ⚡ **Fast & Smooth API Integration**  
- 🎨 **Gradient UI with Smooth Animations**

---

## 🚀 Live Experience

> Type → Translate → Listen → Copy → Switch Theme  

All interactions are designed to feel **fast, responsive, and smooth**

---

## 🧠 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling, Animations, Glass UI  
- **JavaScript (Vanilla JS)** – Core Logic  
- **MyMemory API** – Translation Engine  
- **Web Speech API** – Voice Output  

---

## ⚙️ Core Functionalities

- Real-time DOM manipulation  
- Async API handling using `fetch` & `async/await`  
- Clipboard API integration  
- Speech synthesis (text-to-speech)  
- Theme switching with dynamic UI updates  
- Interactive feedback using toast notifications
